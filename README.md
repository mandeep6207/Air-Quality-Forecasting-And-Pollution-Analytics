An end-to-end data science project that analyses air quality across Indian cities, predicts the **Air Quality Index (AQI)**, and classifies pollution severity using machine learning.

---

## рџ“Њ Project Overview

India's air pollution crisis affects hundreds of millions of people. This project uses daily air quality measurements from the **Central Pollution Control Board (CPCB)** to:

- Understand pollution patterns across cities and seasons
- Predict the numerical AQI value (regression)
- Classify the AQI severity bucket (classification)
- Identify the most influential pollutants

---

## рџ“‚ Dataset Description

| Attribute | Detail |

| File | `data/city_day.csv` |
| Records | 29,531 daily observations |

| Date range | 2015 вЂ“ 2020 |
| Features | PM2.5, PM10, NO, NO2, NOx, NH3, CO, SO2, O3, Benzene, Toluene, Xylene |
| Targets | `AQI` (continuous), `AQI_Bucket` (6 categories) |

**AQI Bucket categories:** Good В· Satisfactory В· Moderate В· Poor В· Very Poor В· Severe

## рџ’Ў Key Insights

1. **PM2.5 is the dominant predictor** of AQI вЂ” its concentration alone explains most of the variance.
2. **Delhi, Patna, and Lucknow** are consistently the most polluted cities in the dataset.
3. **Winter months (NovemberвЂ“January)** show the highest AQI due to crop residue burning and temperature inversions.
4. **Tree-based ensembles** outperform linear models by a large margin (RВІ ~0.97 vs ~0.75).
5. The classifier achieves >95% accuracy on extreme categories (Good / Severe) but shows some confusion between adjacent buckets.
